VascularSim

Open-source cardiovascular simulator

VascularSim is an opensource software for cardio-vascular simulation developed by Touch Lab – IIT Madras.

This software has been developed in close association with distinguished medical institutions to serve as an interactive learning platform for Medical students.

One of the first simulation software devleoped over a HYBRID Cardiovascular model, simulating accurate pressure, flow and volume waveforms.

Customize parameters
User set values for Artery properties such as elastance, resistance and inductance.
Perform Experiments
Simulate multiple experiments and study effects. Models continiously added.
Analyze Waveforms
Analyze the waveforms and calculate pressure, flow, velocity and other values.
